# PROYECTO SWITCHBOT

Este proyecto tiene como objetivo desarrollar un switchbot usando una ESP32 C3 para automatizar<br>
cualquier tarea que requiera el uso de algun interruptor.

## Materiales

- ESP32 C3 zero mini
- Servo Motor SG90
- Dos Baterias 18650
- Dos portapilas para 18650
- Led RGB
- Resistencia 330 ohms
- Resistencia 1 kohm
- Regulador LM7805
- Push Bottom
- Transistor 2N2222A
